# FullStack Upload App

A complete full-stack application that allows users to upload images or text files with a React frontend and FastAPI
backend.

## Features

- 📁 Upload images (JPEG, PNG, GIF, WebP)
- 📝 Upload text content
- 🎯 Clean and responsive UI
- ⏳ Loading states and error handling
- 🐳 Dockerized for easy deployment

## Tech Stack

### Backend

- FastAPI
- Python 3.11
- Uvicorn

### Frontend

- React 18
- Vite
- Axios

### Infrastructure

- Docker
- Docker Compose
- Nginx
